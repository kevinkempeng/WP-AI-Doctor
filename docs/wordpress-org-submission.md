# WordPress.org Submission

This project is prepared for submission as **PressCare AI Error Doctor 1.0.2**.

## Submission files

- Upload `dist/presscare-ai-error-doctor-1.0.2.zip` for initial review.
- Keep the files in `.wordpress-org/` ready for the WordPress.org SVN `assets/` directory after approval.
- Request the slug `presscare-ai-error-doctor` and verify it before accepting the final directory name.

## Before uploading

1. Sign in at https://wordpress.org/plugins/developers/add/ with the account that should own the plugin.
2. For an official PressCare listing, use an account with current contact information and a verified PressCare email address.
3. Confirm that the submitted display name is `PressCare AI Error Doctor`.
4. Upload only the installable plugin ZIP, not the combined submission packet or the asset archive.
5. Watch for mail from `plugins@wordpress.org`, including the spam or junk folder.

## After approval

1. Check out the assigned WordPress.org SVN repository.
2. Copy the release files into `trunk/`.
3. Copy the PNG files from `.wordpress-org/` into the SVN `assets/` directory.
4. Copy the release files into `tags/1.0.2/`.
5. Verify that `Stable tag: 1.0.2` matches the plugin header version and SVN tag.
6. Commit only when the public release is ready; the first SVN commit makes the plugin live.
