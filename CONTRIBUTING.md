# Contributing

Contributions should preserve the plugin's evidence-first, read-only architecture.

## Requirements

- WordPress 7.0 or newer.
- PHP 8.0 or newer.
- WordPress Coding Standards.
- Unit tests for parser, grouping, or redaction changes.
- No real credentials, private logs, customer names, domains, email addresses, IP addresses, or proprietary code in fixtures.

## Pull requests

Explain the problem, the deterministic evidence used, privacy impact, tests added, and any change to data sent externally. AI-generated suggestions must be reviewed and understood by the contributor before submission.

Automated fixes, file editing, shell execution, plugin deactivation, and update installation are outside the version 0.1 scope.
