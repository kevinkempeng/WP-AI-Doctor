# Security policy

Please do not disclose a suspected vulnerability, credential, or production log in a public issue.

Report security concerns privately through https://presscare.com/plugins and include:

- The affected plugin version.
- The WordPress and PHP versions.
- Reproduction steps using synthetic or fully sanitized data.
- The expected and observed behavior.

PressCare AI Error Doctor is read-only by design. A report that the plugin changed code, configuration, plugins, themes, users, or content should be treated as a high-priority security issue.
